# 🏆 JJ Sports

Este repositório contém o código-fonte do site oficial da **JJ Sports**:
👉 https://www.jjsports.com.br/

---

## 🚀 Sobre o projeto

O projeto foi desenvolvido com foco em performance, responsividade e experiência do usuário, atendendo às necessidades de um e-commerce moderno no segmento esportivo.

---

## 🛠️ Tecnologias utilizadas

* **PHP 7.5**
* **Tailwind CSS**
* **HTML5**
* **JavaScript**

---

## 🎯 Objetivo

Criar uma plataforma robusta, rápida e escalável para apresentar a proposta da JJSports:

* Alta performance
* Layout responsivo (mobile-first)
* Facilidade de manutenção
* Boa experiência de navegação
